# ChezAntaBada — Frontend

E-commerce de foulards haut de gamme. Vite + React + Tailwind.

## Stack
- Vite + React 18
- Tailwind CSS
- Framer Motion
- Stripe
- React Query

## Installation

```bash
npm install
cp .env.example .env
# Remplir les variables dans .env
npm run dev
```

## Build

```bash
npm run build
```
